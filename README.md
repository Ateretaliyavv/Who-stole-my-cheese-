# Who-stole-my-cheese- עטרת ון לואן וטליה ולרשטיין
**משחק מירוץ מאתגר, מי יצליח לאסוף את הגבינה ראשון?**

## מהות המשחק

משחק תחרות בין 2 שחקנים המתנהל במבוך.
על 2 השחקים למצוא מקומות גבינה בתוך המבוך ע"י מפת עזר, להצליח לשמור עליה מפני גניבת עכברים ע"י התגוננות עם מלכודת עכברים.
מי שמצליח לאגור ראשון את כמות הגבינה הרצויה באותו שלב, ינצח.
המשחק מתאים למחשב רגיל ולפלאפון
